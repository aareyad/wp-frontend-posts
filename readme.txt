Allow your registered members to post from the frontend.

In the WordPress admin area you can specify the following:

*   Minimum Title Length
*   Minimum Content Length
*   Exclude BuddyPress admin member in member page
*   Show theme info in admin footer

**Usage:**

Add a page with the following shortcode to create a form:

[wfp_frontend_form]
